# English Janala 🇬🇧🪟

**English Janala** is a modern, interactive web application designed to help Bengali speakers learn English effectively through structured lessons, vocabulary practice, and an intuitive user interface.

**Live Demo:**  
🌐 https://rakib-dhali.github.io/English-Janala/

## ✨ Features (Implemented / In Progress)

- Clean and modern UI with **Tailwind CSS + DaisyUI**
- Responsive design (mobile-first)
- Lesson / vocabulary browsing
- Search functionality for quick word or lesson lookup
- Custom fonts and icons integration
- Interactive lesson highlighting
- Level-based learning progression
- Pronunciation support / audio (if added later)

## 🛠️ Tech Stack

- **HTML5**
- **CSS3** 
- **JavaScript**
- **Tailwind CSS** + **DaisyUI**
- **Google Fonts**
- **Font Awesome Icons**
